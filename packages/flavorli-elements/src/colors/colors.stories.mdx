# Colors

<h2>Primary Color</h2>
<p>
The primary color is the main brand color.

To create contrast between UI elements, you can use light or dark variants of
the primary color on each elements. You can also use variants to distinguish
elements within a component, such different variants used on a button, and the
icon within it. </p>
